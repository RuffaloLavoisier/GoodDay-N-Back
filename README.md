# What is that ?  

## Intro  

`We grow up when we rest properly.`  
  
문제적 남자에서 나온 <N-back> 게임에 흥미를 느껴 제작하였습니다.  
프로그래밍 하다가 휴식을 취할 때 사용하여 자투리 시간도 즐겁게 보내고자 제작하였습니다.

## Usage

랜덤하게 나온 숫자가 지정한 N번째 전 숫자와 동일한지 판단합니다.  
동일하면 'O' 아니면 'X' 를 선택해 게임을 진행하는 간단한 게임입니다.  

### Sample

>input : 3,3,5,2,3  

>2-Back : x  
>3-Back : O  
>4-back : O
